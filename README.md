# Devcon Schedule

## How to run the project

### Development

```bash
npm i
npm run dev
```

### Production build

```bash
npm i
npm run build
npm run start
```

## Live deployment

[demo](https://devcon-schedule.vercel.app/)

## Stack used

- `React` + `NextJS` + `TypeScript`
- `ChakraUI` for components UI
- `Zustand` for server state caching
